## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- Java 17+
- MySQL 8.0+
- Angular CLI
- Maven

### Installation
1. Clone the repository
2. Set up the backend:
   ```bash
   cd backend
   mvn install
   ```
3. Set up the frontend:
   ```bash
   cd frontend
   npm install
   ```

### Development
- Backend: `mvn spring-boot:run`
- Frontend: `ng serve`
